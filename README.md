# Mobile Chat UI

A pixel-perfect mobile chat interface built as an interactive UI experiment — bridging the gap between design and code.

**[→ View Live Demo](https://mobile-chat-lemon.vercel.app)**

## Overview

This project is a fully coded mobile chat UI, translated directly from a design concept into a working React component. The goal was to explore how micro-interactions and layout precision can make a familiar pattern — messaging — feel polished and intentional.

## Built With

- **React** — component-based UI structure
- **Vite** — fast dev build tooling
- **Tailwind CSS** — utility-first styling
- **General Sans** — typography via Fontshare

## What I Was Exploring

- Faithful design-to-code translation on mobile viewports
- Smooth, natural-feeling chat bubble layouts
- Clean component structure for a real-world UI pattern
